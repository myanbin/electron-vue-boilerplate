# photos-library

## Project setup

```sh
yarn
# 如果 electron 安装失败，请尝试以下的命令
ELECTRON_MIRROR="https://npm.taobao.org/mirrors/electron/" yarn add electron --dev
```

### Compiles and hot-reloads for development

```sh
yarn electron:serve
```

### Compiles and minifies for production

```sh
yarn electron:build
```
